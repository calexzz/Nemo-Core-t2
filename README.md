# Система планирования рабочих смен

Приложение для хакатона t2 — индивидуальный график сотрудников.

## Стек

- **Backend:** Python 3 + Flask
- **База данных:** SQLite
- **Frontend:** HTML / CSS / JS (на основе шаблона index.html)
- **Экспорт:** openpyxl (Excel .xlsx)

## Установка и запуск

```bash
# 1. Установить зависимости
pip install flask openpyxl

# 2. Запустить приложение
python app.py
```

Открыть браузер: [http://localhost:5000](http://127.0.0.1:5000)

## Функциональность

### Сотрудник
- Авторизация по логину/паролю
- Добавление смен на 2 недели вперёд
- Выбор начала и конца смены (шаг 15 минут)
- Отметка выходного дня
- Редактирование и удаление смен
- Просмотр в виде списка и сетки (2 недели)
- Статистика: кол-во смен, будни/выходные, часов
- Проверка: не более 6 смен подряд, не более 12 часов в день

### Менеджер / Администратор
- Матрица смен всех сотрудников по датам
- Фильтрация по альянсу и группе
- Список сотрудников с кол-вом смен
- Выгрузка в Excel (.xlsx) с разбивкой по дням
- Управление пользователями (только admin): добавление, удаление

## Структура файлов

```
app.py              # Flask маршруты и логика
database.py         # Инициализация БД и вспомогательные функции
schedule.db         # SQLite база (создаётся автоматически)
templates/
  ├── login.html      # Страница входа
  ├── employee.html   # Интерфейс сотрудника
  └── manager.html    # Интерфейс руководителя
```

## API эндпоинты

| Метод | URL | Описание |
|-------|-----|----------|
| GET/POST | /login | Авторизация |
| GET | /schedule | График сотрудника |
| POST | /shift/add | Добавить смену |
| POST | /shift/delete/<id> | Удалить смену |
| POST | /shift/edit/<id> | Редактировать смену |
| GET | /manager | Панель менеджера |
| GET | /export/excel | Выгрузка в Excel |
| GET | /api/users | Список пользователей |
| POST | /api/users/add | Добавить пользователя |
| POST | /api/users/delete/<id> | Удалить пользователя |
